# Student Management System (Java)

This is a simple Java console-based project that I built while learning the basics of Java programming.

## About the Project

This program allows the user to add student names and view the list of students through a simple menu. It runs in a loop until the user chooses to exit.

The main purpose of this project was to practice core programming concepts and understand how a basic application works.

## Features

- Add student names
- View all added students
- Simple menu-based interaction

## Technologies Used

- Java
- VS Code

## How to Run

1. Compile the program:
   javac Main.java

2. Run the program:
   java Main

## What I Learned

- How to use arrays in Java
- Working with loops and conditions
- Taking user input using Scanner
- Building a simple console-based application
